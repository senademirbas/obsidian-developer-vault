# Welcome to Obsidian Developer Vault

This workspace is a personal knowledge and project management template customized for developers. Everything is pre-configured to organize your notes, track your projects, and store code snippets.

---

## Folder Structure

The folder structure in the left sidebar is as follows:
* **00_Dashboard/**: Main dashboard displaying your task calendar and daily progress.
* **01_Daily/**: Daily notes. Automatically created from calendar or shortcut.
* **02_Projects/**: Your active projects and Kanban boards.
* **03_Knowledge/**: Hierarchical tree of learned technologies and concepts.
* **04_Snippets/**: Repository of annotated code snippets.
* **05_Resources/**: Articles, books, PDFs, and other assets.

---

## Basic Workflow Tips

1. **Quick Task Capture (Ctrl+X):**
   * Press Ctrl+X inside any note. Type the task description in the input box, press Enter, then select a due date.
   * Your task will be automatically inserted with the #task tag and due date.

2. **Automatic Modified Date:**
   * The modified field in the frontmatter of your notes automatically updates to the current date (YYYY-MM-DD) as you edit.

3. **Developer Tag Highlights:**
   * The #task and #bug tags are automatically styled as forest green and dark red capsules in the Obsidian editor.

---

## Configuring Plugins
* **Git Backup:** The obsidian-git plugin is disabled by default to prevent startup errors. To backup your notes to GitHub, enable it in Settings -> Community Plugins and connect your own repository. Detailed instructions are in the main GitHub README.
